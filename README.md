1) Fire fighter monitering system
   An application for fire fighters to get the real time locations and alert of the area which they serve.
   Build by using node js framework with mysql database to manage the fire fighter, people, location data for real-time.
   Short polling wad used manitain the real-time constraint.
   twilo communication was used to alert the fire fighter and people in fire accident is recognized.
2) A desktop application was build to recognize the fire via surveilance cameras using pyhton with the help of Tkinter module and OpenCV.
   app.py in the application folder the source code of used Oops to design the application in notebook style.
   This application register the location and the people of that area for allerts to the server.
   This application recognize the fire by its color and movement , for that HSV_masking and frame differniation(used mean square error) to differniate the frame of video to determine the fluctuations in the image which concludes the fire.
